
# Employee Status Enum

The status of the Employee being retrieved.

## Enumeration

`EmployeeStatusEnum`

## Fields

| Name | Description |
|  --- | --- |
| `ACTIVE` | Specifies that the employee is in the Active state. |
| `INACTIVE` | Specifies that the employee is in the Inactive state. |

## Example

```php
use SquareConnectAPILib\Models\EmployeeStatusEnum;

$employeeStatus = EmployeeStatusEnum::ACTIVE;
```
