
# Create Customer Card Response

Defines the fields that are included in the response body of
a request to the `CreateCustomerCard` endpoint.

Either `errors` or `card` is present in a given response (never both).

## Structure

`CreateCustomerCardResponse`

## Fields

| Name | Type | Tags | Description | Getter | Setter |
|  --- | --- | --- | --- | --- | --- |
| `card` | [`?Card`](../../doc/models/card.md) | Optional | Represents the payment details of a card to be used for payments. These<br>details are determined by the payment token generated by Web Payments SDK. | getCard(): ?Card | setCard(?Card card): void |
| `errors` | [`?(Error[])`](../../doc/models/error.md) | Optional | Any errors that occurred during the request. | getErrors(): ?array | setErrors(?array errors): void |

## Example

```php
use SquareConnectAPILib\Models\Builders\CreateCustomerCardResponseBuilder;
use SquareConnectAPILib\Models\Builders\CardBuilder;
use SquareConnectAPILib\Models\Builders\AddressBuilder;
use SquareConnectAPILib\Models\Builders\ErrorBuilder;

$createCustomerCardResponse = CreateCustomerCardResponseBuilder::init()
    ->card(
        CardBuilder::init()
            ->billingAddress(
                AddressBuilder::init()
                    ->addressLine1('500 Electric Ave')
                    ->addressLine2('Suite 600')
                    ->addressLine3('address_line_34')
                    ->administrativeDistrictLevel1('NY')
                    ->administrativeDistrictLevel2('administrative_district_level_26')
                    ->country('US')
                    ->locality('New York')
                    ->postalCode('10003')
                    ->build()
            )
            ->bin('bin6')
            ->cardBrand('VISA')
            ->cardType('card_type8')
            ->cardholderName('Amelia Earhart')
            ->expMonth(11)
            ->expYear(2018)
            ->id('icard-card_id')
            ->last4('1111')
            ->build()
    )
    ->errors(
        [
            ErrorBuilder::init(
                'category8',
                'code8'
            )
                ->detail('detail6')
                ->field('field4')
                ->build(),
            ErrorBuilder::init(
                'category8',
                'code8'
            )
                ->detail('detail6')
                ->field('field4')
                ->build()
        ]
    )
    ->build();
```
