
# Loyalty Event Source Enum

Defines whether the event was generated by the Square Point of Sale.

## Enumeration

`LoyaltyEventSourceEnum`

## Fields

| Name | Description |
|  --- | --- |
| `SQUARE` | The event is generated by the Square Point of Sale (POS). |
| `LOYALTY_API` | The event is generated by something other than the Square Point of Sale that used the Loyalty API. |

## Example

```php
use SquareConnectAPILib\Models\LoyaltyEventSourceEnum;

$loyaltyEventSource = LoyaltyEventSourceEnum::SQUARE;
```
