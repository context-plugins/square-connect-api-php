
# Inventory Adjustment Group

## Structure

`InventoryAdjustmentGroup`

## Fields

| Name | Type | Tags | Description | Getter | Setter |
|  --- | --- | --- | --- | --- | --- |
| `fromState` | `?string` | Optional | Representative `from_state` for adjustments within the group. For example, for a group adjustment from `IN_STOCK` to `SOLD`,<br>there can be two component adjustments in the group: one from `IN_STOCK`to `COMPOSED` and the other one from `COMPOSED` to `SOLD`.<br>Here, the representative `from_state` for the `InventoryAdjustmentGroup` is `IN_STOCK`. | getFromState(): ?string | setFromState(?string fromState): void |
| `id` | `?string` | Optional | A unique ID generated by Square for the<br>`InventoryAdjustmentGroup`.<br><br>**Constraints**: *Maximum Length*: `100` | getId(): ?string | setId(?string id): void |
| `rootAdjustmentId` | `?string` | Optional | The inventory adjustment of the composed variation.<br><br>**Constraints**: *Maximum Length*: `100` | getRootAdjustmentId(): ?string | setRootAdjustmentId(?string rootAdjustmentId): void |
| `toState` | `?string` | Optional | Representative `to_state` for adjustments within group. For example, for a group adjustment from `IN_STOCK` to `SOLD`,<br>the two component adjustments in the group can be from `IN_STOCK` to `COMPOSED` and from `COMPOSED` to `SOLD`.<br>Here, the representative `to_state` of the `InventoryAdjustmentGroup` is `SOLD`. | getToState(): ?string | setToState(?string toState): void |
