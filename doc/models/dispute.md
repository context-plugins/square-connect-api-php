## Example

```php
use SquareConnectAPILib\Models\Builders\DisputeBuilder;
use SquareConnectAPILib\Models\Builders\MoneyBuilder;

$dispute = DisputeBuilder::init()
    ->amountMoney(
        MoneyBuilder::init()
            ->amount(186)
            ->currency('currency8')
            ->build()
    )
    ->brandDisputeId('brand_dispute_id6')
    ->cardBrand('card_brand0')
    ->createdAt('created_at4')
    ->disputeId('dispute_id8')
    ->build();
```
